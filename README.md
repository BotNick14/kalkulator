ПРОЕКТ: КАЛКУЛАТОР

 СЪДЪРЖАНИЕ:
- models.py: Логически клас Calculator с методи за обработка на данни.
- main.py: Интерактивен интерфейс за терминала с безкраен цикъл (while-loop).
- 
ИНСТРУКЦИИ:
1. Отворете терминал/команден ред в папката на проекта.
2. Стартирайте приложението:
   python main.py
3. Следвайте инструкциите в менюто на екрана.
   
ФУНКЦИОНАЛНОСТ:
- Поддържа +, -, * и /.
- Защита от деление на нула и невалидни символи.
- Памет за сесията (история).
- Статистика за натовареността.
